void SinglyList::add ( int  data)

First sets the size parameter in order to call the new and delete operators in Node.cpp. Then it calls the new operator, receiving either a recycled node or a newly created node, and it sets its data to the data in the parameter. Finally, it checks if the list is empty, if it is, it maes that node the head, otherwise it pushes the first to the second position and sets the new node as head. Finally it prints the list.
